# 🔧 Bot WhatsApp Barbearia - Versão Simplificada

Bot automatizado para barbearia via WhatsApp. Sem banco de dados, apenas arquivos JSON.

## 🚀 Funcionalidades

✅ Agendamento rápido de cortes  
✅ Horários disponíveis por barbeiro  
✅ Confirmação instantânea  
✅ Lista de preços  
✅ Funciona 24/7

## 📦 Instalação

```bash
git clone https://github.com/rickzadaxzl-crypto/barbershop-bot.git
cd barbershop-bot
pip install -r requirements.txt
python bot.py
```

## 🎯 Como Usar

1. Execute `python bot.py`
2. Escaneie o QR code com WhatsApp
3. Pronto! Bot funcionando

## 📝 Comandos

- **oi** - Menu principal
- **1** - Agendar
- **2** - Horários
- **3** - Barbeiros
- **4** - Preços
